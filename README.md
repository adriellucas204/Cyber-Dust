Hello, that's the repository from a game made by me, it was a project for the Object Oriented Programming course. There are bugs and is necessary an ASCII Engine, made by the professor from the course, in order to run, therefore, this repository only contains the source code and the assets used in the game.
Implemented in C++ and the pillars of Object Oriented Programming were used, Inheritance, Abstraction, Encapsulation and Polymorphism.
